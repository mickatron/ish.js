The tween factory

ish.tween(options)

Description:

Tweens values between two points to be used in Javascript animation. Animation works a little differently to jQuery's animat() method, this simply provides callback functions that fires on a timer and provides parameters for that tweens frames animation values. You can then write your own CSS manipulation routine within the callbacks.

Returns
ish.tween

Chainable, returns the module instance.

Parameters:
Name Type Description
options Object
Properties
Name Type Description
from Number

The start value of the tween.
to Number

The end value of the tween.
easing String

Specify an easing equation for the tween.
onBeforeTween function

A callback function fired before each tween begins.
onTween function

A callback function fired each tween frame.
onComplete function

A callback function fired when the tween is complete.
Example:
var animateOpacity = $.tween({
  from: 1,
  to: 0,
  duration: 1000,
  easing: 'linear',
  onBeforeTween: function (){
  	//onBeforeTween operations
  },
  ontween: function (value){
  	ish('.selector').css('opacity', value)
  },
  oncomplete: function (){ 
  	//onComplete operations
  }
}).start();

Methods

this.nextFrame(frameNumber)

line 149
Description:

Moves the tween to the next frame or the specified interval.

Returns
ish.tween

Chainable, returns the module instance.

Parameters:
Name Type Description
frameNumber Int

The frame number you wish to skip to.
Example:
animateOpacity.start();
animateOpacity.stop();

// move to the next frame
animateOpacity.nextFrame();
// skip to a specific tween frame
animateOpacity.nextFrame(500);

this.stop(skipToEnd)

line 191
Description:

Stops the tween.

Returns
ish.tween

Chainable, returns the module instance.

Parameters:
Name Type Description
skipToEnd Boolean

Set to true to end the tween.
Example:
// stop the animation at the current frame.
animateOpacity.stop();
// stop the animation and skip to the end of the tween.
animateOpacity.stop(true);
